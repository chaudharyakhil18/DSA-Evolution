# Prefix Sum 

Linked list problems focus on pointer manipulation, structure transformations, and cycle detection.

Key topics:

- Reversing a linked list (entire list or sublists).
- Detecting and removing cycles (Floyd's cycle detection).
- Merging, splitting, and reordering lists.
- Using dummy nodes to simplify pointer updates.
